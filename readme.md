/**********************************
 * jmeter mongodb插件
   
/***************************************


**使用方法：**<br />
1.clone工程到本地。<br />
2.使用maven打成jar包。<br />
3.copy jar包到 jmeter安装目录 \lib\ext 下。<br />
4.运行jmeter, 在线程组下添加java request 选择对应工程中的类名即可（前提是先要保证mongodb可用）<br />

本工程插件在jmeter 5.4.1版本下测试通过
